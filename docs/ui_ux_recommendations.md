# UI/UX рекомендации

## Наблюдения
- Основной геймплей понятен: крупное поле, контрастные фигуры, быстрые переходы между экранами.
- Есть разделение на Play, Daily, Themes и Settings без перегруженности интерфейса.
- Экран результатов уже содержит ключевые CTA: Play Again, Menu и rewarded-кнопки.

## Рекомендации и быстрые улучшения
- Держать явную обратную связь по rewarded-кнопкам:
  - disabled-состояние;
  - короткие подсказки по причине недоступности: cooldown, недостаточный score или tokens.
- На тач-устройствах сохранять `tap-to-place` как fallback при неточном drag.
- Проверять safe-area на мобильных устройствах, особенно верхний HUD и нижние CTA в портретном режиме.
- Для доступности:
  - не полагаться только на цвет для подсветок;
  - сохранять достаточный контраст текста в карточках и кнопках.

## Дизайн-направление
- Поддерживать визуальный стиль Lume: мягкое свечение, чистые формы, минимум визуального шума.
- Анимации делать короткими и функциональными: line-clear flash, feedback на действия игрока.
- Избегать всплывающих элементов, которые перекрывают игровое поле во время активного хода.
